Tutorial

SUPER APP

¡Bienvenido a SuperApp!

El juego abarca múltiples áreas para desarrollar el reconocimiento, gestión y compra de los productos necesarios en el día a día dentro de los supermercados.


Al presionar el botón de Volumen se activan y desactivan todos los sonidos de la aplicación. Este botón se encuentra en la parte superior derecha del menú principal y también en todas las pestañas dentro del juego de SUPERMERCADO y el juego de PALABRAS


Al presionar el botón de Usuarios, se despliegan otros dos botones; uno para Crear usuario y otro para Ingresar, Este botón se encuentra en la parte inferior derecha del menú principal.


• El botón Crear usuario, desplegado hacia arriba, sirve para crear un nuevo usuario. Se puede elegir un avatar, tomar una foto o eligir una imagen de la galería del dispositivo para el icono del nuevo usuario y se le debe asignar un nombre en el espacio “Usuario”.


• El botón Ingresar, desplegado hacia la izquierda, sirve para ingresar con los usuarios que fueron creados escribiendo el nombre de alguno en el espacio “Usuario”.


Al presionar el botón de Información, se despliegan otros dos botones; uno para ver este tutorial y otro para ver información acerca de nuestro proyecto. Este botón de encuentra en la parte inferior izquierda del menú principal.


Al presionar el botón Editar accederás a la lista de productos registrados del usuario actual donde puedes:


• Agregar un nuevo producto presionando el botón “Nuevo Item” en el cual, al presionarlo, luego podrás escoger una categoría, un nombre, una imagen, un audio y si estará o no Activo dentro de los juegos.


• Administrar las CATEGORIAS presionando esta palabra en la esquina superior derecha en la cual luego podrás editar o eliminar una categoría existente y crear una NUEVA CATEGORIA.


• Escuchar el sonido de cada producto, editarlo y marcarlo/desmarcarlo como Activo para los juegos de la aplicación


La aplicación tiene el juego de SUPERMERCADO y el juego de PALABRAS.


Al presionar sobre el icono de SUPERMERCADO aparecerá el botón de una LISTA en el cual se podrán agregar productos de cada una de las categorías desde el almacén arrastrándolos hacia la lista y luego se podrá guardar esta, o ver una lista que fue anteriormente guardada con los iconos en la parte superior derecha. Una vez guardada la lista aparecerá otro icono para borrarla, Esta funcionalidad sirve para imitar listas de compras reales.


Al volver a presionar sobre el icono de SUPERMERCADO aparecerán las distintas dificultades del juego para empezar a jugar.
